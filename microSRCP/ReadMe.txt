
Sourcen fuer Arduino IDE und Beispieldatei RocRail.

Arduino IDE
-----------
skeetbook-ZIP-Datei entpacken und Inhalte in Arduino Skeetbook-
Verzeichnis kopieren. (siehe File -> Preferences in der 
Arduino IDE).

Mittels File -> Sketchbook koennen dann die einzelne
Board Sketches geoeffnet, compiliert und ins Board
geladen werden.

RocRail
-------
Die SRCP Zentrale kann einfach mittels dem User-Interface von 
RocRail erstellt werden.
Dazu ist eine neue BiDiB Zentrale einzutragen und bei der 
Kommunikation Serial ohne CTS einzutragen.

Die .xml Dateien sind ausgesuchte Beispiele welche
in microSRCP.odt beschrieben sind.
